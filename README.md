# A simple personal blog

## Developed using Node.js, MongoDB, Express, and EJS

### Author: Chang Zhao 1925828

#### User guide

1. Install packages

  ```bash
  npm install
  ```

2. Run server

  ```bash
  nodemon server.js
  ```

3. Access website with address 

  ```url
  127.0.0.1:5000
  ```
